# 大乐透+排列三彩票预测

排列三 + 大乐透统一桌面预测工具。两个玩法**分区独立**：各自算法、各自历史数据、各自权重，互不影响。

已录入近100期中奖号码 需要手动导入新的中奖号码

> **声明**：本软件仅供学习、统计与算法研究。彩票开奖为随机事件，任何模型都无法保证中奖。请理性购彩，勿沉迷。

## 玩法说明

### 排列三（直选）

- 三位数字 `0–9`，**顺序必须完全一致**  
- 例：开奖 `1 2 4` → 1号位=1，2号位=2，3号位=3  

### 大乐透

- **前区**：01–35 任选 5 个（顺序无关）  
- **后区**：01–12 任选 2 个（顺序无关）  

---

## 算法说明

两端均为 **多算法加权集成（Ensemble）**：各位/号码层打分 → 组合成注 → 综合排序输出 TopN。  
开奖录入并「对比上次预测」后，按各算法在本期 Top10 上的表现 **EMA 修正权重**（表现好的算法权重大，差的变小）。

### 排列三算法

**各位打分（Digit）**

| 算法 | 思路简述 |
|------|----------|
| 加权频率 | 历史出现频率，近期加权更高 |
| 逐位马尔可夫 | 同一位置相邻期转移概率 |
| Softmax 回归 | 对近期特征做 soft 分类打分 |
| 遗漏冷热 | 遗漏期数与冷热号平衡 |
| EMA 趋势 | 指数滑动平均捕捉短期走势 |
| AR 自回归 | 简单自回归外推 |
| 贝叶斯融合 | 先验 + 似然融合 |
| 间隔规律 | 号码间隔分布规律 |
| 迷你 RNN | 轻量序列模型对各位打分 |

**组合打分（Combo）**

| 算法 | 思路简述 |
|------|----------|
| 各位融合分 | 三位分数合成一注 |
| 整注马尔可夫 | 整注层面转移/共现 |
| 形态匹配 | 奇偶、大小、跨度等形态 |
| 蒙特卡洛 | 随机抽样估计组合强度 |

### 大乐透算法

**号码层（适合无序组合）**

| 算法 | 思路简述 |
|------|----------|
| 加权频率 | 历史频率 + 近期加权 |
| 遗漏冷热 | 遗漏与冷热号 |
| EMA 趋势 | 短期热度趋势 |
| 贝叶斯融合 | 贝叶斯平滑 |
| 间隔规律 | 开出间隔统计 |
| 共现关联 | 号码两两同开关联 |

**组合层**

| 算法 | 思路简述 |
|------|----------|
| 号码融合分 | 单号分合成前区/后区组合 |
| 形态匹配 | 区间、奇偶、和值等形态 |
| 共现加分 | 组合内共现关系加分 |
| 集合马尔可夫 | 集合状态转移 |
| 冷热组合分 | 组合整体冷热均衡 |

> 两套算法**互不共享权重与历史**，切换分区不会串数据。

## 目录结构

```
lottery_suite/
├── app_gui.py          # 统一桌面界面
├── build_exe.bat       # 一键打包 exe
├── pl3/
│   └── predict.py      # 排列三算法引擎
├── dlt/
│   └── predict.py      # 大乐透算法引擎
└── data/
    ├── pl3/            # 排列三历史、权重、上次预测等
    └── dlt/            # 大乐透历史、权重、上次预测等
```

数据文件（运行后自动维护）：

- `history.json` — 开奖历史  
- `algo_weights.json` — 算法权重  
- `last_prediction.json` — 上次预测 Top10（用于对比）  
- `compare_log.json` — 对比与修正日志  
- `model.json` — 模型缓存

## 运行方式

### 环境

- Windows 10/11  
- Python 3.10+（推荐 3.12）  
- 标准库 + `tkinter`（一般随 Python 安装）  
- 大乐透 Excel 导入需要：`pip install openpyxl`  
- 打包需要：`pip install pyinstaller`

### 源码运行

```bash
cd lottery_suite
python app_gui.py
```

### 打包 exe

```bash
# 双击或在命令行执行
build_exe.bat
```

完成后：

- `dist/LotterySuite.exe`  
- `dist/彩票号码助手.exe`  
- 数据目录：`dist/data/pl3`、`dist/data/dlt`

## 使用流程（简要）

1. 选择分区（排列三 / 大乐透）  
2. **录入 / 删除**：录入历史开奖（期号 + 号码）  
3. **预测**：点击「开始预测」，查看 TopN 与推荐注  
4. 下一期开奖后录入号码，点「对比上次预测」→ 自动按算法表现修正权重  
5. 可随时「重置算法权重」恢复默认
